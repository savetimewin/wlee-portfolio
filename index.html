Zap

Description:
Zap is minimalist survival shooter built with Python, using the Pygame library. You pilot a neon pink circle in an 800×600 arena, dodge procedurally spawned squares that constantly come toward the screen center, and fire 8‑way lasers with WASD/ZQECX inputs to clear space. The game tracks how long you last, ramps spawn rate and speed as minutes pass, and offers a simple start screen plus a “Try Again?” game-over loop so you can chase longer survivals.

Implementation:

  • Implemented the core game loop and state flow (start screen → gameplay → game over → retry).
  • Built procedural enemy spawning with simple “home toward center” movement and difficulty ramp over time.
  • Added player movement, 8‑way shooting, projectile/enemy updates, and collision detection for hits + death.
  • Created basic UI elements (start button, game-over prompt) and an on-screen survival timer.

Future implementation consider for the improvement of the game:

  • Add a point system where taking fewer steps to dodge awards points or less penalty — rewards efficient movement and encourages planning instead of constant strafing.
  • Add a center boundary area that grants more points — nudges players toward higher-risk positioning that forces sharper dodges.
  • Slow down the spawn rate and speed of squares and have them bounce back when hitting the edge — smooths difficulty spikes and keeps pressure without edge-stacking.
  • Add a leaderboard to see who had most points and longest survival time, and add your own records — gives replayability and a measurable goal for skill progression.
